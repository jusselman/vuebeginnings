Getting Started with Vue.js

Below is the button with 'Show Alert' will trigger an alert window.
This is done by attaching the 'v-on:click' interpolation to the button.
after the onclick listener, we assign the function 'alertWarning' which
is defined in Javascript.
JS -
new Vue ({
el: '#exercise',
data: {
value: ''
},
methods: {
alertWarning: function() {
alert('You have been warned');
}
}
});

Next, below is a counter which will present us with the x and y axis values when moving mouse
over the area of the element, except the 'dead-zone' where the rule is not applicable due tothe
'span v-on:mousemove.stop="">Dead Spot' JS - new Vue({
el: '#app',
data: {
counter: 0,
x: 0,
y: 0
},
methods: {
increase: function(step) {
this.counter += step;
},
updateCordinates: function(event) {
this.x = event.clientX;
this.y = event.clientY;
},

}
}
